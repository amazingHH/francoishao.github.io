# 🛠 Projects

## 🔋 PLC over Battery Cells (PhD Core Project)

Investigating the feasibility of using battery cells as a communication medium using powerline communication techniques.

**Work includes:**
- Designing PLC hardware interface for battery systems
- Measuring attenuation and bit error rate (BER)
- Studying effect of battery chemistry and SOC on signal quality

---

## 📡 VNA-Based Signal Characterisation Setup

Built a measurement setup using SMA-modified PCB interfaces to analyse impedance and signal propagation in electrochemical systems.

---

## ⚡ Battery Cycling Data Acquisition System

Developed a system for long-term cycling data collection to correlate electrochemical state with communication channel quality.

---

## 🧪 NI LabVIEW Control System

Automated control of NI instruments for repeatable PLC testing and measurement logging.
